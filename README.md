# My Silverstripe Vue Project

This is a web app using Silverstripe CMS and Vue.js.

## Setup

1. Install PHP and Composer.
2. Run `composer install` in the root directory.
3. For the frontend, go to frontend/ and run `npm install`, then `npm run serve`.

## Running

- Start Silverstripe: Use the Silverstripe dev server or configure web server.
- Vue: `npm run serve` in frontend/.
